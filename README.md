# Школа Высшей Математики "Бесконечность"

Это репозиторий для лендинга школы "Бесконечность", созданного в рамках демонстрации навыков веб-разработки и тестирования.

## Описание

Лендинг представляет собой яркую, современную веб-страницу с 3D-эффектами, посвящённую школе высшей математики. Он предназначен для привлечения потенциальных учеников (школьников и студентов), предоставляет информацию о курсах и способах связи, а также содержит форму регистрации.

## Технологии

*   **HTML5**: Структура страницы.
*   **CSS3**: Стилизация, анимации, адаптивность.
*   **JavaScript (ES6+)**: Логика взаимодействия, работа с формой.
*   **Three.js**: Создание и анимация 3D-объектов на фоне.

## Особенности

*   Яркий неоновый дизайн с темно-синим фоном.
*   Анимированные 3D-фигуры, плавающие по краям экрана.
*   Четкое разделение информации: "О Нас", "Курсы", "Контакты", "Форма записи".
*   Ссылки на внешние ресурсы (обучение, YouTube, соцсети).
*   Адаптирован для просмотра на различных устройствах (в процессе).
